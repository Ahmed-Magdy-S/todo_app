# Task App API

> Backend API for Task App.

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev
# Run in prod mode
npm start
```

## Database Seeder

To seed the database with users, tasks data from the "\_data" folder, run

```
# Destroy all data
node seeder -d
# Import all data
node seeder -i
```
